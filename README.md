# Xbox DNS Switcher

Простой скрипт и набор ярлыков для быстрого переключения системного DNS на **xbox-dns.ru** (176.99.11.77, 80.78.247.254).

## Возможности
- **Включить (`don` / Ярлык):** Устанавливает DNS xbox-dns.ru и запрещает получение DNS от роутера.
- **Выключить (`dof` / Ярлык):** Сбрасывает настройки на автоматические (DHCP от роутера).
- **GUI:** Добавляет удобные ярлыки в меню приложений и на панель.

## Установка
Запустите `install.sh`:
```bash
chmod +x install.sh
./install.sh
```

## Использование
- **Терминал:**
  - `don` — Включить
  - `dof` — Выключить
- **Графический интерфейс:**
  - Найдите "Xbox DNS ON" или "Xbox DNS OFF" в меню.
